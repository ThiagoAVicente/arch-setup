# Keybindings

`$mainMod` = `SUPER` (Windows key)

## Programs

| Keybind | Action |
|---------|--------|
| `SUPER + Return` | Terminal (foot) |
| `SUPER + D` | App launcher (rofi) |
| `SUPER + E` | File manager (thunar) |
| `SUPER + Q` | Close window |
| `SUPER + F` | Fullscreen |
| `SUPER + T` | Toggle floating |

## Window Management

| Keybind | Action |
|---------|--------|
| `SUPER + Arrows` | Move focus |
| `SUPER + SHIFT + Arrows` | Move window |
| `SUPER + LMB drag` | Move window |
| `SUPER + RMB drag` | Resize window |
| `SUPER + S` | Pin window |

## Workspaces

| Keybind | Action |
|---------|--------|
| `SUPER + 1-6` | Switch to workspace 1-6 |
| `SUPER + G` | Switch to workspace 7 |
| `SUPER + M` | Switch to workspace 8 |
| `SUPER + SHIFT + 1-6` | Move window to workspace 1-6 |
| `SUPER + SHIFT + G` | Move window to workspace 7 |
| `SUPER + SHIFT + M` | Move window to workspace 8 |
| `SUPER + Scroll Up/Down` | Switch workspaces |

## Media & System

| Keybind | Action |
|---------|--------|
| `Volume Up/Down` | Adjust volume ±5% |
| `Mute` | Toggle mute |
| `Mic Mute` | Toggle microphone |
| `Brightness Up/Down` | Adjust brightness ±10% |
| `Play/Pause` | Media play/pause |
| `Next/Previous` | Next/previous track |
| `SUPER + Tab` | Spotify play/pause |

## Custom Commands

| Keybind | Action |
|---------|--------|
| `CTRL + SHIFT + S` | Screenshot (area select) |
| `SUPER + SHIFT + V` | Toggle screen recording |
| `SUPER + L` | Lock screen |
| `SUPER + B` | Toggle waybar |
| `SUPER + Escape` | Power menu |
| `SUPER + ALT + W` | Change wallpaper |
| `SUPER + N` | Toggle gammastep (blue light) |
| `SUPER + P` | Toggle debug mode |
| `SUPER + SHIFT + R` | Reload Hyprland |

## Resize Mode

| Keybind | Action |
|---------|--------|
| `SUPER + R` | Enter resize mode |
| `Arrows` (in resize mode) | Resize active window |
| `Return/Escape` | Exit resize mode |

## Zoom

| Keybind | Action |
|---------|--------|
| `SUPER + Plus` | Zoom in |
| `SUPER + Minus` | Zoom out |
| `SUPER + SHIFT + Plus/Minus` | Reset zoom |

## Gestures

| Gesture | Action |
|---------|--------|
| `3 fingers horizontal` | Switch workspaces |
| `3 fingers vertical` | Resize window |
| `4 fingers horizontal` | Fullscreen |
