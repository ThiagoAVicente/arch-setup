# Post-Installation

After running `setup.sh`, do this:

## 1. Download Wallpapers

```bash
# Download wallpapers to this directory
~/Pictures/Wallpapers/
```

## 2. Select Your Wallpaper

Press `SUPER + ALT + W` and select a wallpaper.

This will automatically:
- Set the wallpaper
- Generate color scheme
- Apply theme to all apps

## 3. Done!
